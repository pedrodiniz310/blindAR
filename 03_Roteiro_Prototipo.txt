1. O QUE É O PROTÓTIPO
================================================================================

Um web app interativo que simula a experiência do SecureVision usando a webcam
do notebook ou câmera do smartphone como substituto do HoloLens 2.

Demonstra os 4 pilares:
  1. Autenticação facial contínua
  2. Consulta a IA com filtro de segurança
  3. Mudança de nível contextual em tempo real
  4. Dashboard de governança

O protótipo é gravado em vídeo de 2 minutos para o Entregável 2.

Arquivo do protótipo: prototipo_securevision.html
Para rodar: abrir no navegador (Chrome/Edge) — não precisa instalar nada.


2. FUNCIONALIDADES DO PROTÓTIPO
================================================================================

2.1 TELA DE LOGIN / AUTENTICAÇÃO FACIAL
────────────────────────────────────────
  • Webcam ativa ao abrir a página
  • Detecção facial via face-api.js (TensorFlow.js)
  • Ao detectar rosto → "Identidade verificada"
  • Mostra: nome do usuário, cargo, nível de segurança calculado
  • Se rosto sai do frame → "SESSÃO TRAVADA" em < 1 segundo
  • Se rosto retorna → desbloqueia automaticamente

2.2 INTERFACE AR (ASSISTENTE DE IA)
────────────────────────────────────
  • Simula a visão do óculos AR (HUD transparente)
  • Campo de texto para perguntar ao assistente de IA
  • Respostas são filtradas pelo nível de segurança atual
  • Watermark dinâmico com nome + hora no canto da tela
  • Perguntas pré-configuradas para demonstração:
    - "Qual a pressão atual do poço P-47?"
    - "Mostre os dados de produção do mês"
    - "Quem é o responsável pela plataforma?"

2.3 PAINEL DE NÍVEL DE SEGURANÇA
─────────────────────────────────
  • Barra lateral mostra o nível atual (1 a 5) com cor
  • Botões para simular mudança de contexto:
    - "Rede Interna" → Nível 1 (verde)
    - "Rede Corporativa" → Nível 2 (azul)
    - "Rede de Terceiro" → Nível 3 (amarelo)
    - "Área Pública" → Nível 4 (laranja)
    - "Dispositivo Comprometido" → Nível 5 (vermelho)
  • Ao mudar nível, a resposta da IA muda em tempo real
  • Demonstração visual: mesma pergunta, respostas diferentes

2.4 DETECÇÃO DE OBSERVADOR
──────────────────────────
  • Se a câmera detectar MAIS DE UM rosto → alerta
  • "Observador detectado — nível elevado para 4"
  • Dados sensíveis ficam borrados automaticamente
  • Volta ao normal quando observador sai do frame

2.5 DASHBOARD DE GOVERNANÇA
────────────────────────────
  • Aba separada no protótipo
  • Mostra log de eventos em tempo real:
    - Autenticações (sucesso/falha)
    - Consultas à IA (pergunta + nível aplicado)
    - Mudanças de nível
    - Detecções de observador
    - Alertas de segurança
  • Gráficos:
    - Distribuição de acessos por nível
    - Timeline de eventos
    - Status dos dispositivos ativos

4. TECNOLOGIAS DO PROTÓTIPO
================================================================================

  • HTML5 + CSS3 + JavaScript (vanilla)
  • face-api.js — detecção facial no navegador (não envia dados)
  • API simulada — respostas pré-configuradas por nível
    (não precisa de API real para a demo)
  • Chart.js — gráficos do dashboard
  • Funciona offline — apenas abrir o .html no navegador
